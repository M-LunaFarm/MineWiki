# MineWiki Legacy Sources

This directory keeps migration references only.

- `legacy/mwiki-fastify` is not an active production runtime.
- `legacy/luna-votifier` is not an active production runtime.
- Active MineWiki code lives under `apps/*`, `packages/*`, and `prisma/*`.

Do not start, deploy, or add build scripts for legacy applications. Move required behavior into the integrated monorepo before using it in production.
